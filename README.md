# Projeto-IA-Fiocruz-Chatbot
O IA-Fiocruz-Chatbot é uma solução avançada de IA baseada em RAG, desenvolvida para consulta precisa a documentos institucionais da Fiocruz. Utiliza Next.js 15, Azure OpenAI (GPT-4o), AI Search e Cohere Rerank para fornecer respostas fundamentadas com citações em tempo real, garantindo segurança via Azure AD e alta performance.
